# Excel 多关键词筛选工具

一个纯前端（浏览器本地处理）的 Excel 多关键词筛选与编辑工具，替代 WPS「高级筛选」繁琐的多步操作。**数据全程不上传服务器**。

## ✨ 功能特性

- **多关键词筛选**：选定筛选列，输入多个关键词（空格 / 逗号 / 换行均可分隔），一键筛出所有包含行
- **三种匹配模式**：包含任一（OR）/ 包含全部（AND）/ 不包含任一（排除）
- **单元格编辑**：双击任意单元格直接编辑，回车保存、Esc 取消；支持删除行、新增行、撤销全部编辑
- **高亮选中值**：单击单元格高亮同列所有相同值，快速查看数据分布
- **冻结窗格**：可选冻结表头 / 表头 + N 行，滚动时顶部固定便于对照
- **双模式导出**：导出筛选结果 / 导出完整表格（保留编辑）
- **实时预览**：输入即筛选，命中关键词高亮（黄色）、筛选列高亮（绿色）
- **多工作表 & 表头开关**：支持多 Sheet 切换、首行是否为表头
- **深浅色主题** & 响应式布局（移动端可用）
- **隐私优先**：基于 SheetJS，文件解析与导出全部在浏览器本地完成

## 🛠 技术栈

- [Next.js 16](https://nextjs.org/) (App Router) + TypeScript 5
- [Tailwind CSS 4](https://tailwindcss.com/) + [shadcn/ui](https://ui.shadcn.com/)
- [SheetJS (xlsx)](https://docs.sheetjs.com/) —— Excel 解析与导出
- [lucide-react](https://lucide.dev/) 图标 + [sonner](https://sonner.emilkowal.ski/) 通知

## 📖 使用说明

1. **上传表格**：拖拽或点击上传 `.xlsx / .xls / .csv` 文件（也可点「加载示例数据」体验）
2. **设置筛选条件**：选定筛选列 → 输入关键词（空格/逗号/换行分隔）→ 选择匹配方式
3. **筛选与编辑**：
   - **双击**单元格 → 编辑（回车保存、Esc 取消）
   - **单击**单元格 → 高亮同列相同值
   - 行末 **🗑** → 删除该行；底部 **+ 新增一行**
   - 顶部 **撤销全部编辑** → 还原原始数据
4. **冻结窗格**：表格上方下拉选择「仅冻结表头 / 表头 + N 行」
5. **导出**：导出筛选结果 / 导出完整表格（均为 `.xlsx`，保留编辑与数据类型）

## 🔒 隐私说明

所有文件处理（解析、筛选、编辑、导出）完全在浏览器本地进行，**文件内容不会上传到任何服务器**。关闭页面即清除。
